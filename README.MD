## ⚙️ Tecnologias utilizadas:

- EsLint;
- Prettier;
- Yarn;
- React Native;
- Styled Componemts;
- Expo;
- React Navigation;

## 🏁 Como rodar o Projeto:

Clone o repositório:

```bash
$ git clone https://github.com/araujooleticiaa/pomodora
```
<br/>

cd `pomodora` e rode:

```bash
yarn
```
Para iniciar o mobile:

```bash
yarn start
```
